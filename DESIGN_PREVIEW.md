# Codec Plugin - 新设计预览

## 🎨 设计理念
基于 CyberChef 的功能设计，采用现代深色技术工作台风格，注重：
- 视觉层次清晰
- 色彩搭配专业
- 交互反馈流畅
- 信息密度适中

## 🌟 主要改进

### 1. 色彩系统
- **主色调**: 紫蓝渐变 (#6366f1 → #8b5cf6)
- **背景层次**: 深蓝灰色系，营造专注的技术氛围
- **状态颜色**: 成功/警告/错误色彩明确

### 2. 布局优化
- **左侧工具库**: 固定宽度，分类清晰，搜索便捷
- **中间操作链**: 中央工作区，拖拽区域明显
- **右侧输入输出**: 独立面板，结果展示清晰

### 3. 交互增强
- 悬停效果：卡片上浮、边框高亮
- 拖拽反馈：虚线边框、区域高亮
- 动画过渡：流畅的淡入效果
- 状态指示：成功/错误视觉反馈

### 4. 细节打磨
- 渐变顶部指示条
- 左侧彩色标记
- 圆角阴影处理
- 滚动条样式统一
- 响应式布局适配

### 视觉特色
1. **深色主题**: 护眼且专业，适合长时间使用
2. **渐变强调**: 紫蓝渐变作为品牌识别色
3. **层次分明**: 不同背景深度区分功能区域
4. **微交互**: 悬停、点击、拖拽都有视觉反馈
5. **技术美学**: 字体、间距、色彩都体现技术工具特点
